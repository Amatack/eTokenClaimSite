# ETOKEN CLAIM APP

## Deploy

Deploy the client on your server by executing the following commands:

npm run build  
npm run preview

Deploy the server folder on your server by executing the following command:

npm run start

Configure the following environment variables

On Client:

VITE_URL_API=

On Server:

ACTIVE=  
ALLOWED_BY_CORS=  
DATABASE=  
DB_URI=  
CHRONIK_INSTANCE=  
MNEMONIC=  
SENDER_ADDRESS=  
TOKEN_ID=  
AMOUNT_OF_ETOKEN=  
AMOUNT_OF_XEC=
